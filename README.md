# 🎮 DEADZONE — FPS Survival Shooter

<img width="1200" height="900" alt="collage (1)" src="https://github.com/user-attachments/assets/d78e032f-a41c-4495-9626-d82708c10276" />




A modern browser-based FPS survival shooter built using HTML, CSS, JavaScript, and Three.js.  
Fight endless enemy waves, survive as long as possible, and test your shooting accuracy in an immersive survival arena.

---

## 🚀 Features

- 🔫 FPS Survival Gameplay
- 🌊 Endless Wave Mode
- 🎯 Real-time Shooting Mechanics
- ❤ Health System
- 🔄 Reload System
- 💀 Enemy Counter & Kill Feed
- 🖱 Mouse Aim + Pointer Lock Controls
- ⏸ Pause Menu
- 📊 Accuracy Tracking
- 🎨 Stylish HUD & UI Effects
- ⚡ Three.js Powered Graphics

| Key | Action |
|------|--------|

| `SPACE` | Jump |
| `LMB` | Shoot |
| `R` | Reload |
| `MOUSE` | Aim |
| `ESC` | Pause |
